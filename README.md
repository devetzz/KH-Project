# JAVA-Project
JAVA_영화 예매 프로그램


## 🖥️ 프로젝트 소개
JAVA의 기본 기능들로 구현한 영화 예매 프로그램<br>
DB를 배우기 전에 파일을 통해 데이터 관리<br>

Oracle 연동 방식 학습 후 DB 연동 버전 추가 (DBConnect)

## 🕰️ 개발 기간
* 파일 버전 : 25.04.23일 - 25.04.26일
* DB 버전 : 25.06.09일 - 25.06.13일

### ⚙️ 개발 환경
- `Java`
- `JDK 24`
- `VisualStudio Code`
- `Oracle`
